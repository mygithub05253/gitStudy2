버전 : 프로그램이나 파일이 수정된 사항이 있으면 버전이 업데이트 되었다.

버전 관리 시스템 : git의 수많은 버전을 쉽게 관리하도록 도와주는 도구

git
1. 분산형 버전 관리 시스템
=> 원격 저장소(github)와 로컬 저장소(개인 컴퓨터)
=> 인터넷 연결이 없더라도 작업 수행 가능
2. 원격 저장소에 있는 코드 공유 가능
=> 효율적인 협업 가능
3. 파일들의 수정, 삭제, 추가 등에 대한 모든 변경 사항들이 기록됨
4. 오픈소스를 보며 코드의 지식을 넓히는 것 가능

윈도우에서 설치 방법
: 구글 git 검색 -> download for windows -> click here to download
-> 설치 파일 실행 -> 기본 설정 유지한 채로 Next

git 시작
1. 로컬레포지토리 설정
2. cmd 창 혹은 git bash here
=> 레포지토리로 만들고자 하는 폴더 안에서 명령어 입력 필수!
3. git init
=> .git이라는 숨김 폴더가 만들어지면서 해당 폴더를 로컬 저장소로 사용 가능
=> 최초 한 번만 실행해주면 됨
=> .git 폴더를 삭제한 경우 다시 로컬 저장소로 설정하기 위해서 git init 재실행
